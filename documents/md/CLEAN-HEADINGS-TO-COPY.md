## How to Replace in Word:

### Method 1: Direct Edit
1. Go to the heading in your document
2. Select the heading text
3. Delete it
4. Copy the clean heading from above
5. Paste it in place

### Method 2: Find & Replace (if "%3" is visible in text)
1. Press `Ctrl + H`
2. Find: `7.3.%3` (or whatever shows in your doc)
3. Replace: `7.3` (just the number)
4. Click "Replace All"

### Method 3: Update Heading Style
1. Right-click the heading in navigation pane
2. Select "Modify Style"
3. Check the formatting
4. If you see "%3" in the style name, remove it

## Note:
The "%3" is **NOT in the source files** - all markdown files are clean. This is purely a Word display/formatting issue. After replacing with these clean headings, the "%3" should disappear from the navigation pane.
