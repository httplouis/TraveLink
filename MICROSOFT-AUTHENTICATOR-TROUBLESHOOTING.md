## Problem: Walang Code sa Microsoft Authenticator

Kung walang notification o code na lumalabas sa Microsoft Authenticator app, try these solutions:

---

## ‚úÖ Solution 1: Check App Notifications

### Steps:
1. **Open** Microsoft Authenticator app sa phone
2. **Check** kung may pending notifications
3. **Swipe down** to refresh
4. **Look** for "Enverga University" or "Microsoft" notification

### If No Notification:
- **Check** phone settings - dapat enabled ang notifications for Microsoft Authenticator
- **Check** if phone is connected to internet
- **Try** opening the app manually

---

## ‚úÖ Solution 2: Use "Sign in another way"

### Steps:
1. **Click** "More information" link (blue text sa page)
2. **Look** for "Sign in another way" option
3. **Select** alternative method:
   - Email code
   - SMS code
   - Phone call

### Alternative Methods:
- **Email code** - Code will be sent to your email
- **SMS code** - Code will be sent via text message
- **Phone call** - Automated call with code

---

## ‚úÖ Solution 3: Check Authenticator App Settings

### Steps:
1. **Open** Microsoft Authenticator app
2. **Go to** Settings
3. **Check** if account is properly added:
   - Should see "Enverga University" or your email
   - Should be active (not expired)

### If Account Not Added:
1. **Add account** manually:
   - Tap "+" or "Add account"
   - Select "Work or school account"
   - Scan QR code or enter manually

---

## ‚úÖ Solution 4: Request New Code

### Steps:
1. **Wait** 30 seconds
2. **Refresh** the page
3. **Try** entering code again
4. **Or** click "Resend code" if available

---

## ‚úÖ Solution 5: Use Different Device

### Steps:
1. **Try** on different device (laptop, tablet, phone)
2. **Or** use different browser
3. **Or** use incognito/private mode

---

## üì± Common Issues

### Issue 1: App Not Receiving Notifications
**Fix:**
- Check phone notification settings
- Ensure app has notification permissions
- Try manual refresh in app

### Issue 2: Code Expired
**Fix:**
- Request new code
- Refresh the page
- Try again

### Issue 3: Wrong Account
**Fix:**
- Verify you're using the correct Microsoft account
- Check if account is added in Authenticator app
- Re-add account if needed

### Issue 4: Time Sync Issue
**Fix:**
- Check phone time is correct
- Sync time in phone settings
- Restart Authenticator app
